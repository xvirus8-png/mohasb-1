# مساعد شخصي بالذكاء الاصطناعي - Project Context

## Startup Protocol
في بداية كل محادثة جديدة:
1. اقرأ هذا الملف (الذاكره.txt) تلقائياً
2. اقرأ آخر جلسة من ملخص المحادثات في أسفل الملف
3. حدّث ملخص المحادثات بعد كل جلسة

## عن المشروع
- **الاسم:** مساعد شخصي AI (مساعدي المحاسبي)
- **الوصف:** تطبيق محاسبي ذكي - يحلل الجمل العربية ويسجلها في Excel على Google Drive
- **المجلد:** `C:\Users\medo2\Desktop\مساعد شخصي AI`
- **تاريخ البدء:** 15 يونيو 2026
- **التبعية:** مشروع مستقل، لا علاقة له بمشاريع GOOGLE APP

## المكدس التقني
- **الواجهة:** HTML + CSS + Vanilla JS (PWA)
- **التخزين:** Google Drive API (.xlsx) + LocalStorage (كاش)
- **AI:** Google Gemini API (مجاني) + تحليل محلي (بدون API)
- **Excel:** SheetJS (xlsx.js) لقراءة وكتابة xlsx
- **التصميم:** Dark/Light mode + RTL + شاشتين

## القواعد الأساسية (البونت)
1. كل محادثة جديدة اقرأ هذا الملف أولاً
2. كل التعديلات والملفات الجديدة تكون داخل مجلد هذا المشروع فقط
3. لا تتدخل في مشاريع GOOGLE APP إلا بطلب مباشر
4. احفظ نسخة باك أب قبل أي تعديل كبير
5. وثّق كل قرار معماري في الذاكره.txt
6. اشرح الكود بطريقة بسيطة
7. لا تتحرك بخطوة قبل ما يفهم المستخدم
8. **كل تعديل يصير بملف جديد وفيرجن جديد** — ما نكتب فوق الملف القديم أبداً. نحتفظ بنسخ الفيرجنات السابقة.

## البروتوكولات

### بروتوكول التخطيط (قبل أي تنفيذ)
1. حلّل المتطلبات والقيود
2. حدد افتراضاتك بوضوح
3. إذا فيه غموض، اسأل المستخدم فوراً
4. اقترح الحل الأبسط (Simplicity First)
5. ناقش الخطة مع المستخدم
6. نفّذ فقط بعد الموافقة

### بروتوكول التنفيذ
1. الكود يكون كاملاً وجاهزاً (ممنوع الـ TODO أو Placeholder)
2. عالج الأخطاء (error handling)
3. اختبر الشغل قبل لا تسلمه
4. نظف المخلفات (ما تترك أثر)
5. حدّث الذاكره.txt بعد كل إنجاز

### بروتوكول التعديل الجراحي
1. المس فقط ما يجب لمسه - لا تحسن كود مجاور
2. التزم بأسلوب الكود الحالي حتى لو مو مثالي
3. إذا تسبب تعديلك في جعل دالة يتيمة، احذفها
4. أقل تعديل ممكن يحل المشكلة

### بروتوكول التشخيص والإنقاذ
1. **ممنوع التخمين** - اجمع الأدلة أولاً
2. حاول تعيد إنتاج الخطأ
3. تتبع الخطأ من نقطة الانهيار للمصدر
4. أصلح بأقل تعديل ممكن
5. اكتب اختبار يمنع تكرار المشكلة
6. أزل أكواد التشخيص المؤقتة
